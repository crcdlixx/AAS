# Release Notes

## Unreleased

- 修复生产环境“打开白屏”：移除 Vite 手动分包（`manualChunks`）以避免循环依赖导致的运行时错误。
- 解题模式从“全局选择”改为“每道题单独选择”（`自动 / 单模型 / 双模型`）。
- 文理分科从“自动判断”改为“按页选择文科/理科”；未选择分科时会阻止解题并提示。
- 后端支持前端按页提交 `subject`（优先按用户选择路由；未提供 `subject` 时才使用 `ROUTER_*` 自动兜底）。
- 修复跨图合并时可能丢失题型提示（如“多选/不定项”要求）导致 AI 误判单选：跨图合并会携带上一页最后题目的整个合并组。
- 强化选择题兜底提示：要求模型先判断单选/多选/不定项；多选/不定项最终答案必须输出全部正确选项（如 ACD）。
- 部署体验优化：
  - Nginx 提升上传大小限制（避免较大图片上传失败）。
  - Caddy 增加 `/foo/assets/*` → `/assets/*` 重写，兼容 `base: './'` 的相对资源路径在子路径访问时不被 SPA fallback 误伤。
